SKILDR

senior project
spring 2023
student project

Skildr was created for a senior capstone project, combining all the skills learned through the Graphic Design and Interactive Media degree. It’s an original concept that utilizes AI technology to curate resources and create lesson plans based off a given skill. Currently a WIP.
It’s broken down into three components. Branding design, which consists of a logo, posters, and stationary deliverables. Second is the UX/UI concept design for a Skildr application. Third is a web based prototype that utilizes the OpenAI API in order to create a functioning prototype.
